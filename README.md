puppet-duplicity
================
[![Build Status](https://travis-ci.org/mgruener/puppet-duplicity.png?branch=master)](https://travis-ci.org/mgruener/puppet-duplicity)

**Manages duplicity**

This module is used to manage the backup tool duplicity. This does not include performing
backups with duplicity, for this see my puppet-backup module. For the moment this means
puppet-duplicity only install duplicity.
